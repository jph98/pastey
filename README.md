pastey
======

Pastebin built with:
* Slim - PHP micro REST framework
* Ractive - Javascript data binding

Make sure you install the following package managers:
* JS (Bower) - npm install -g bower
* PHP (Composer) - curl -sS https://getcomposer.org/installer | php
